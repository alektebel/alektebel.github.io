## Custom domain

Yes — GitHub Pages supports custom domains. To use one:
1. Add a `CNAME` file at the root of this repository containing your domain (e.g. `yourdomain.com`).
2. Point your domain's DNS to GitHub's servers (see [GitHub docs](https://docs.github.com/en/pages/configuring-a-custom-domain-for-your-github-pages-site)).
3. Enable the custom domain in the repository's **Settings → Pages** section.
